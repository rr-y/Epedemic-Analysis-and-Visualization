# Simulation and Visualization of Real Time Epidemic Spreading 

## Requirement
	Python 3.x
	Django Version == 1.9
	Google Map Api Key
	
## How to run?
	Go to the project folder and type --
	1- python3 manage.py runserver
	2- open your browser and type
	3- http://127.0.0.1:8000/map/index
